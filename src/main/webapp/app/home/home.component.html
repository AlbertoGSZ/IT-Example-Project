Base de datos de información criminal

Bienvenid@

Esta aplicación permite la gestión de perfiles criminales, así como la relación entre los mismos, sus antecedentes, pruebas y registros carcelarios, ayudando con ello a conocer el historial delictivo y las relaciones entre sujetos.

Para comenzar haz click en las pestañas de Operaciones y Administración.

- Administrador (usuario="admin" y contraseña="admin")
- Usuario estándar (usuario="user" and contraseña="user").
